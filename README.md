# UseMon 📊

UseMon is a lightweight Windows screen time tracker that monitors application usage and helps you understand how you spend your time on your computer.

## 🚀 Features
- Track active application usage
- Measure time spent on each app
- Runs silently in the background
- Simple and efficient

## 🛠️ Tech Stack
- Python (pywin32, psutil)
- JSON / SQLite (for storage)
- HTML, CSS, JavaScript (dashboard - upcoming)

## 📌 Future Plans
- Website tracking (Chrome Extension)
- Visual dashboard with charts
- Daily/weekly reports
- Productivity classification
- Focus mode (block distracting apps)

pip install pywin32 psutil
